Plataforma

Proporciona tres grados de libertad, logrando movimiento vertical, giro sobre el eje vertical y giro sobre el eje de profundidad. Esta configuración presenta un grado de complejidad mayor para su control debido a que los tres actuadores deben ser controlados simultáneamente para cada uno de los movimientos que se realizan. Así mismo los tres actuadores deben de ser capaces de soportar el peso de la estructura y del usuario.

Plataforma

Una vez elegido el modelo para la elaboración de la plataforma se procedió a la creación de un render de cómo operaría la plataforma usando el modelo de 3 grados de libertad.

Escenario

Dentro de los productos de software que se desarrollaron se encuentran principalmente modelos tridimensionales que sirven para representar el ambiente virtual, para la generación de dichos modelos se realizó una investigación para saber si existe un estándar en cuanto a medidas y ubicación del mobiliario urbano.
